PB_ImageTo3D User Guide

PB_ImageTo3D converts normal 2D image files into interactive pseudo-3D presentations with layered parallax, a depth mesh, exportable layer assets, and a smoke-test workspace designed for quick validation.

Quick Start

Fast path

  1. Launch PB_ImageTo3D.exe.
  2. Open a JPG, PNG, GIF, BMP, TGA, or TIFF image.
  3. Move the mouse inside the viewport.
  4. Switch between Parallax View, Depth Mesh, and Smoke Test.
  5. Optionally load foreground and middle masks.
  6. Export generated layers if you want to inspect the results.

Best source images

  • portraits with clean subject separation
  • objects photographed against simpler backgrounds
  • landscapes with obvious foreground and horizon depth
  • artwork with strong value contrast between layers
The app can auto-generate masks, but manual grayscale masks almost always produce cleaner depth separation.

Recommended Workflow

Step 1

Load an image

Begin with a source image. The app resizes very large images internally so texture generation stays practical.

Step 2

Check generated previews

Review the front, middle, and depth previews. If they do not look believable, tune the threshold sliders or load masks.

Step 3

Refine interaction

Adjust depth strength, sensitivity, smoothing, camera distance, and mesh strength until motion feels intentional.

Step 4

Export and save

Export PNG layers, then save a project file so you can return to the exact same setup later.

Viewer Modes

The program now supports three ways to inspect the same source image.

Parallax View
Depth Mesh

Parallax View

Creates separate back, middle, and front planes that shift and rotate at different rates. This is usually the most stable and readable pseudo-3D mode.

Depth Mesh

Pushes a subdivided plane forward and backward using the generated depth map, creating a more sculpted appearance.

Smoke Test

Uses a dual-view layout so you can compare parallax and mesh behavior at the same time while validating the build.

Masks and Depth Generation

Masks control which image regions are treated as foreground or middle distance. The remaining content becomes part of the back layer and also contributes to the depth map.

Input Purpose What to paint white
Foreground Mask Pushes the most important subject forward person, face, product, main object
Middle Mask Keeps secondary forms between front and back trees, props, furniture, architecture details
Auto Masks Fallback when manual masks are absent generated from brightness and thresholds

Brightness interpretation

  • White means strong inclusion in that layer.
  • Gray means a softer transition.
  • Black means exclusion from that layer.

Auto-mask tuning

  • Auto Front Threshold controls when bright areas begin to feel like foreground.
  • Auto Mid Low and Auto Mid High define the middle-depth band.
  • If the result feels wrong, export the generated depth map and inspect it.

Controls

Depth Strength

Controls layer separation and camera drift in the parallax scene.

Mouse Sensitivity

Scales how strongly mouse movement influences tilt and motion.

Smoothing

Controls how quickly the scene catches up to input. Lower values feel heavier; higher values react more quickly.

Camera Distance

Moves the camera closer or farther away from the scene.

Mesh Strength

Changes how much the depth map deforms the mesh in Depth Mesh mode.

Projects

Project files use .prefs and remember your working state so you can return later without rebuilding the scene manually.

Saved values

  • source image path
  • foreground and middle mask paths
  • depth strength, sensitivity, smoothing, camera distance
  • mesh strength and active viewer mode
  • auto-mask threshold values

Suggested habit

After finding a convincing setup, save a project before exporting layers so you can always reopen the exact same state for follow-up editing.

Recommended Smoke Test

  1. Run the packaged executable.
  2. Load SampleProject.prefs or SampleLandscape.jpg.
  3. Check that the three previews populate.
  4. Verify mouse response in Parallax View.
  5. Switch to Depth Mesh and test Mesh Strength.
  6. Switch to Smoke Test and confirm the dual viewport appears.
  7. Export generated layers.
  8. Save a project and load it back.
  9. Open this help file from the application menu.
If something looks wrong, the previews usually reveal the cause quickly. Check whether the foreground or mid mask is too broad, too narrow, or whether the thresholds need adjustment.

Troubleshooting

The scene looks too flat

  • Increase Depth Strength.
  • Lower the front threshold slightly.
  • Use a stronger manual foreground mask.

The depth mesh looks too extreme

  • Reduce Mesh Strength.
  • Inspect the exported depth map.
  • Tighten the auto-mask range or provide manual masks.

A mask seems ignored

  • Use grayscale input rather than painted color art.
  • Make sure white marks the intended region.
  • Confirm you loaded the correct mask file for the intended layer.

The help file does not open

  • Keep PB_ImageTo3D_Help.html next to the executable.
  • Try opening it manually from Windows Explorer.
  • If the app is moved, move the help file with it.
